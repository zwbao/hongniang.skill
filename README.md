## 架构

```
你的机器                                             服务器
┌──────────────────────────────┐                   ┌──────────────────────────────┐
│                              │                   │                              │
│   私人红娘 (Agent Skill)      │                   │   红娘引擎 (OpenClaw fork)    │
│                              │                   │                              │
│   ┌────────────────────────┐ │    REST API       │   ┌────────────────────────┐ │
│   │ 对话 → 了解你是谁      │ │ ───────────────→  │   │ SQL 预筛 (毫秒)        │ │
│   │ 筛选 → 后端推20个,     │ │                   │   │ 向量粗排 (秒级)         │ │
│   │        我只给你看3个   │ │ ←───────────────  │   │ LLM 精判 (分钟级)       │ │
│   │ 传话 → 帮你问,帮你答  │ │                   │   │                        │ │
│   │ 引导 → 你真正要什么    │ │    反馈闭环        │   │ 自进化: 每日复盘        │ │
│   └────────────────────────┘ │ ───────────────→  │   │ 越用越准               │ │
│                              │                   │   └────────────────────────┘ │
│   对话记录只在这里,不上传     │                   │   只有结构化画像,没有对话     │
│                              │                   │                              │
└──────────────────────────────┘                   └──────────────────────────────┘
```

**核心创新：每个用户都有一位专属的私人红娘。** 她不是后端 API 的壳——她有自己的判断力。

后端匹配引擎基于结构化档案做粗筛，但本地红娘在对话中积累了大量档案里写不出来的信息：你的隐性偏好、回避的话题、前后矛盾的地方。后端推 20 个候选，红娘可能压住 17 个，只展示 3 个她真正觉得合适的。

她的筛选判断会回传给后端，后端从中学习——如果某类推荐总被红娘压住，说明匹配逻辑在那个维度上不准。**用户越多，每位红娘越聪明。**

## 使用

安装后，对你的 agent 说：

```
红娘
```

或者 "我想找对象"、"帮我看看有没有合适的人"。

第一次，红娘会跟你聊天——了解你的工作、生活、性格。你甚至不会意识到自己在"建档"。之后每次来，她会检查有没有新进展、新推荐，像一个真正在帮你操心的人。

## 红娘能做什么

| 功能 | 说明 |
|------|------|
| **对话式了解你** | 通过聊天建立你的画像 |
| **智能筛选推荐** | 后端粗筛 → 红娘细筛，只展示她觉得值得看的 |
| **居间传话** | 双方见面前，红娘在中间传话试探，把握分寸 |
| **婚恋认知引导** | 帮你区分短期吸引和长期兼容，看清自己真正要什么 |
| **约会跟进** | 交换联系方式后跟进，反馈给引擎持续优化 |
| **暂停/恢复** | 在发展一段关系？暂停匹配，想回来随时说 |

---

## 隐私

- 你和红娘的对话**只存在你的本地设备上**，永不上传
- 联系方式只存本地，双方同意后才交换
- 后端只有结构化画像（年龄、城市、性格标签），没有姓名，没有对话
- 随时可以注销，删除所有数据

---

## 红娘的认知框架

红娘不是随便配对的算法。她有自己的婚恋认知体系：

- **关系是长期复利资产** — 短期吸引容易量化但无法预测十年后。品格、价值观和生活节奏的兼容才是长期信号
- **品格高于条件** — 诚实、情绪稳定、善良——这些在长期关系中的权重远大于学历和收入
- **想找到对的人，先成为对的人** — 红娘帮你匹配，也帮你在对话中看清自己
- **伴侣是彼此的雕塑家** — 好的匹配不仅是"合适"，更是"让彼此变得更好"
